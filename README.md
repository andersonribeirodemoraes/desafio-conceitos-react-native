## Instalação e rodando o projeto
```sh
$ git clone https://github.com/andersonribeirodemoraes/desafio-conceitos-react-native.git
$ cd desafio-conceitos-react-native  
$ yarn
$ yarn run android
```

## Rodando os testes
```sh
$ yarn test
```
